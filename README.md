# S.O.N.A.R.
### Sistema de Observabilidad de Nodos y Análisis de Red

PoC de monitoreo de red universitaria Cisco Catalyst usando Python, InfluxDB y Grafana.

## Stack
- Python 3.10+
- Netmiko (conexión SSH)
- InfluxDB 2.x (base de datos de series de tiempo)
- Grafana (visualización)
- Docker Compose
